Ubuntu is one of the most user friendly Debian based Linux distributions. It curently has a LTS version out, Ubuntu 14.04, codename Trusty Tahr. As of the older LTS 12.04(Precise Pangolin) both the server and desktop version will have support for 5 years. Meaning they will have security updates for 5 years. After 5 years it is wise to switch to the newer LTS release.

Now, there are two types of releases, the LTS and regular releases. The LTS releases pops out every 2 years, and most companies only use an LTS release, which aims for stability. The regular releases come out every 6 month, and they usually have the latest features, but in my experience, do not provide the most stable experience.

Security

As with any Linux distribution, security is solid from the kernel up. Does that mean that all Linux distribution are un-hackable? No. But with Linux you're good for 99% of the time. On top of that, the UK security branch have stated: "Ubuntu most secure end-user OS". Ubuntu - and Linux in general - is as transparent as can be. For people that are interested, can view the Ubuntu security notices, for everything security related, affecting the current supported releases of Ubuntu.

Freedom

Since Ubuntu is an open-source operating system, it gives you freedom other proprietary systems don't give you. Ubuntu uses Debian as its base. And many other use Ubuntu as there base. Some have gotten official Ubuntu backing.

Internet

There are three popular web browsers on Ubuntu and Linux in general. One of them is installed in most Linux distros by default Ubuntu is one of those distros, and the browser is Mozilla Firefox. The others are Chrome/Chromium(the open-source version of chrome) and Opera.

Office work

When one has a suit and tie ready for work, he has not completed his big boy image just yet, one needs to make the 'am a big boy image' complete, by using an office application that is equally superb as ones suit, thats where LibreOffice comes in. A superb Office Suite.

LibreOffice is a fully featured suite, with a word processor, spreadsheet and presentation application, that are all Microsoft office compatible. On top of that you also get, an application similar to Microsoft Visio, called LibreOffice Draw. LibreOffice Math lets you edit and create mathematical formulas. Microsoft access users out there, can use LibreOffice base, a database management program. And you can't beat its price: free. For Linux, Mac and Windows.
